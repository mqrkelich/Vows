# Vows - Modern Wedding Planner

**Vows** is a wedding management platform designed to eliminate the chaos of spreadsheets and outdated planning tools. It provides couples with a dashboard to manage guests, design seating layout, and have real time updates for confirmations.

# Key Features
* **Drag-and-Drop Seating Planner:** A interface to arrange guests into tables.
* **Guest Management:** Categorize guests (Family, Friends, Work).
* **Real-Time Analytics:** Live updates of people coming or not and budget breakdowns.
* **Invitation Engine:** For each couple, custom url is provided.
